    What is DooCoins Kids App?

    We created a Kids version of DooCoins with limited features. Children can see their DooCoins balance and transactions, they can request task completion and request to claim rewards. Parents need to approve or decline the requests in the Parent App.

    The Kids version of DooCoins is targetted at tablets, as more children have access to tablets than phones, the app will still function on phones.

    To set your child up on the Kids App, first Install the Kids version, and then invite the child from within the Parents App. See below for more information.

    How do I install DooCoins Kids App?

    See Connect to the Kids App for instructions on how to install the Kids App

    How do I invite a child to use DooCoins Kids App?

    To invite a child to use the kids version of DooCoins, in the Parent App swipe on the child name on the "my children" screen and tap "Invite". A temporary code will be displayed, this code can be used to login to the kids app.

    How do I login to the DooCoins Kids App?

    Open the App on the child's device by clicking the DooCoins icon on the home screen, enter the code that was generated when you invited the child. The code expires, so you might need to generate a new one. The child will stay logged in unless they logout, in which case a new invite code will need to be generated in the Parents App.

    How does DooCoins Kids App work?

    1. Transaction Screen - The Transaction Screen lists historical task and reward transactions.
    2. Task Screen - The Task Screen lists the child's tasks, clicking on the tick icon next to a task will send a task request to the parent's app.
    3. Reward Screen - The Reward Screen lists the child's rewards, if the child has enough DooCoins to claim a reward, clicking on the tick icon next to a reward will send a reward request to the parent's app. If the child doesn't have enough DooCoins for the reward, the a target icon will be displayed, clicking this will set the reward as the child's savings goal.

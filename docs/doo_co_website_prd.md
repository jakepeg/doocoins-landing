# DooCoins Website v2 --- Product Requirements Document (PRD)

## Product Overview

**Product:** DooCoins Website v2\
**Website:** www.doo.co\
**Launch Phase:** Native iOS App Launch\
**Business Model:** Freemium + Family+ (\$2.99/month, Apple Family
Sharing)

### Objective

Redesign the website to: - Reflect updated native app UI and features -
Drive App Store downloads - Clearly communicate value - Convert visitors
into Family+ subscribers - Build trust with parents

------------------------------------------------------------------------

## Goals & Success Metrics

### Primary Goal

Increase App Store installs.

### KPIs

-   Visit → App Store click rate: 15--25%
-   Bounce rate: \<45%
-   Avg time on site: \>1:30
-   Family+ conversion rate
-   Email signups (optional): 5--10%

------------------------------------------------------------------------

## Target Audience

### Primary

Parents aged 30--50 who want: - Less reminding - More structure for
children - Habit-building tools - Positive motivation systems

### Secondary

Families using chore charts or allowance systems seeking digital
solutions.
